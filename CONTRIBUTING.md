## Contributing

Thanks for your interest in contributing! Please follow these guidelines:

1. Fork the repo and create a feature branch.
2. Keep PRs focused and clearly described.
3. Add tests or examples when reasonable.
4. Ensure code is readable and follows the existing style.
5. Link any related issues.

### Local setup
```
python -m venv .venv
.venv\Scripts\activate
pip install -r requirements.txt
python -m src.main
```

### Reporting issues
Provide reproduction steps, expected vs actual behavior, and environment details.
